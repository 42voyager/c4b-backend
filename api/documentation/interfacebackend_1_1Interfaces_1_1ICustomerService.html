Detailed Description

Interface ICustomerService tem os serviços que auxiliam os controllers nas requisições. Para pegar, salvar, atualizar e deletar os dados do banco de dados. E tambem prepara o email para enviar. Tambem posibilita pegar e atualizar os status do customer.
